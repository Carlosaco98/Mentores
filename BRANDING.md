# Coinnecta Mentores Branding

Este proyecto debe seguir el branding de Coinnecta.

Referencia principal: https://www.coinnecta.es/

## Direccion visual

- Base negra/blanca con amarillo Coinnecta como acento principal.
- Interfaz limpia y operativa, sin parecer una landing page.
- CTAs y estados destacados en amarillo.
- Navegacion y marca con fondo negro o carbon.
- Tipografia contundente, con titulares fuertes y texto funcional.
- Nombre de marca preferente: `COINNECTA`.

## Paleta inicial usada

- Negro principal: `#090909`
- Carbon navegacion: `#151515`
- Amarillo Coinnecta: `#f7b500`
- Amarillo hover: `#ffc82e`
- Fondo claro: `#f4f3ef`
- Superficie: `#ffffff`
- Lineas: `#dedbd2`

Si se actualiza la web oficial, revisar esta paleta contra `coinnecta.es` antes de redisenar pantallas.
